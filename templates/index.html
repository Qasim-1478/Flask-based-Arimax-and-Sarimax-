ARIMA

An ARIMA (Autoregressive Integrated Moving Average) model is a statistical tool used for time series forecasting that predicts future values based on past data. It combines three key components: autoregression (AR), which uses past values to predict the future; integration (I), which uses differencing to make the data stationary; and moving average (MA), which uses past prediction errors.

SARIMAX

A SARIMAX model is a statistical time series forecasting model that stands for Seasonal Autoregressive Integrated Moving Average with Exogenous Regressors. It extends the SARIMA model by incorporating external variables, making it a powerful tool for forecasting data that shows seasonality and is influenced by other factors. The model requires specifying parameters for the autoregressive (AR), integrated (I), moving average (MA) components, as well as seasonal and exogenous variables.
